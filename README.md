# 🌌 Astro App - Astronomy Explorer

An interactive web application to explore celestial bodies, learn about astronomy, and discover space phenomena. Built with modern web technologies for an immersive learning experience.

## ✨ Key Features

- 🔭 **Celestial Database** - Browse planets, stars, and galaxies
- 🌍 **3D Visualization** - Interactive 3D models of celestial objects
- 📊 **Planet Explorer** - Detailed information about planets and moons
- 🌙 **Moon Phases** - Track lunar cycles and phases
- ⭐ **Star Catalog** - Search and learn about stars
- 🎨 **Beautiful UI** - Immersive space-themed interface
- 📱 **Responsive Design** - Works on all devices
- 🔍 **Search Functionality** - Find celestial objects easily

## 🛠️ Tech Stack

**Frontend:**
- TypeScript
- React/Web Components
- CSS3 (space-themed animations)
- Interactive 3D graphics

**Features:**
- Real-time astronomical data
- Interactive celestial navigation
- Educational content
- Responsive layout

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/deyjaydeep40-arch/astroapp.git
cd astroapp

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
